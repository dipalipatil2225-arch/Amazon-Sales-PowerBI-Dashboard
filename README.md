# 📊 Amazon Sales Analysis – Power BI Dashboard

## 🔍 Project Overview
This project analyzes Amazon sales data using Power BI to identify top-selling products, category performance, and sales trends.

## 📌 Key Features
- KPI Cards: Total Sales, Total Reviews, Average Price
- Sales analysis by Product Category
- Top-selling products using ranked bar chart
- Interactive slicers:
  - Product Category
  - Shipment Type
  - Order Date

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling
- Data Visualization
